# MotionSlice UI Style Guide

MotionSlice 是一个 Electron 桌面视频工具，不是传统 Web 后台页面。

UI 目标：

暗黑后台系统视觉基因 + 紫蓝主色 + Apple-like 精致感 + 专业视频剪辑工作台。

## 核心视觉继承

必须继承现有后台剪辑系统的视觉基因：

- 暗黑模式
- 紫蓝主色 `#4F39F6`
- 深色背景
- 清晰层级
- 圆角卡片
- 轻量边框
- 克制阴影
- 现代简约字体

## 高级化方向

MotionSlice 要比普通后台页面更高级，更像桌面生产力工具。

应该呈现为：

- 专业视频工作台
- 深色剪辑软件界面
- Apple-like 的精致和克制
- 清晰但不廉价的层级
- 工具感强，但不要复杂混乱

不要做成：

- 普通后台表格页面
- 大面积表单页面
- Web 管理系统首页
- 花哨炫酷大屏
- 纯 Apple 官网风

## 推荐布局

主界面推荐三栏结构：

1. 左侧资源栏
   - 视频文件列表
   - 任务列表
   - 切片列表
   - 宽度 240px 到 280px

2. 中间工作区
   - 视频预览
   - 播放控制条
   - 时间轴
   - 切片轨道
   - 当前播放指针

3. 右侧分析面板
   - 视频信息
   - 晃动识别结果
   - 异常片段
   - 建议裁剪
   - 导出设置
   - 宽度 320px 到 360px

## 页面级约束

- 根节点使用 `.video-tool-page`
- 视频工具相关 class 使用 `vt-` 前缀
- 自定义样式必须限制在 `.video-tool-page` 下
- 颜色优先使用 `src/styles/theme.css` 中的 CSS 变量
- 不要污染全局样式
- 不要大范围覆盖原生元素或第三方组件样式

## 颜色

优先使用 CSS 变量，不要在组件里到处写死颜色。

基础颜色：

- 主背景：`#000000`
- 一级背景：`#0B0B0D`
- 二级背景：`#111116`
- 面板背景：`rgba(26, 26, 30, 0.84)`
- 主色：`#4F39F6`
- 主色 hover：`#5B46FF`

文字颜色：

- 主标题：`#FFFFFF`
- 正文：`#E8E8EA`
- 次级文字：`#A0A0AA`
- 辅助文字：`#777782`

功能色：

- 稳定 / 成功：`#30B08F`
- 轻微晃动 / 警告：`#FEC171`
- 明显晃动 / 危险：`#C03639`
- 信息 / 识别中：`#4AB7BD`
- 重点标记：`#E65D6E`

## 视频预览区

视频预览区是视觉中心。

要求：

- 使用黑色舞台背景
- 视频容器圆角 12px
- 外层使用细边框
- 控制条要自定义，不要裸用 `<video controls>` 作为主视觉
- 播放按钮、暂停按钮、倍速、时间显示要有桌面工具感
- 控制按钮可以使用圆形或胶囊形

## 时间轴

时间轴是核心组件，必须做出专业剪辑工具感。

要求：

- 深色轨道背景
- 切片块使用紫蓝弱背景
- 选中切片使用紫蓝描边和轻微 glow
- 当前播放位置使用细竖线和顶部小三角
- 时间刻度使用低对比文字
- 晃动区间用黄色或红色半透明覆盖
- hover 时展示片段时间、晃动强度、建议动作

禁止：

- 不要把时间轴做成普通表格
- 不要把时间轴做成普通列表
- 不要让时间轴看起来像后台数据卡片

## 晃动识别结果

晃动识别结果必须可视化。

右侧面板至少应支持这些表达：

- 稳定：绿色 tag
- 轻微晃动：黄色 tag
- 明显晃动：红色 tag
- 严重晃动片段：在时间轴上标记
- 总时长
- 晃动片段数
- 最大晃动强度
- 建议裁剪数量
- 可自动处理 / 需人工确认状态

不要只用大段文字描述分析结果。

## 组件风格

按钮：

- 高度 36px 到 44px
- 圆角 10px
- 主按钮使用紫蓝色
- hover 轻微变亮
- active 轻微下压
- 不要使用厚重阴影

卡片：

- 圆角 12px 到 14px
- 深色半透明背景
- 低对比细边框
- 轻微阴影
- 不要每个卡片都做强玻璃效果

输入框：

- 高度 40px
- 深色背景
- 细边框
- focus 使用紫蓝描边
- 不要亮白输入框

标签：

- 圆角 999px
- 小尺寸
- 弱背景 + 彩色文字
- 不要大色块

## 动效

动效要克制：

- hover 160ms 到 220ms
- 面板切换 240ms 到 280ms
- 不要弹跳
- 不要复杂 loading
- 不要花哨渐变动画

## 禁止项

- 禁止做成普通后台表格页
- 禁止大面积白底
- 禁止大面积纯紫色
- 禁止厚重阴影
- 禁止复杂炫彩渐变
- 禁止过度玻璃拟态
- 禁止把所有内容堆进一个大卡片
- 禁止破坏 Electron 进程边界

## 颜色与变量约束 (Strict Variables)

**严禁在 `.vue` 组件的 `<style>` 或行内样式中写死 Hex 颜色值 (如 `#000000`, `#4F39F6`)。**

所有颜色、边框、层级背景必须 100% 使用 `src/styles/theme.css` 中定义的 CSS 变量。
例如：
- 🟢 正确：`background: var(--vt-bg-elevated); border: 1px solid var(--vt-border);`
- ❌ 错误：`background: #111116; border: 1px solid rgba(255,255,255,0.08);`

如果你不确定有哪些变量可用，请先读取 `src/styles/theme.css` 文件。
